# Francisco Guadarrama — Emil UI Libraries CV Site

Static site ready for GitHub Pages.

## Upload
Unzip this folder and upload `index.html`, `style.css`, `script.js`, and `README.md` to the GitHub Pages root.

## Notes
The page is dependency-free for reliable manual upload, but every section is named after Emil Kowalski's recommended libraries and includes working/dummy equivalents: NumberFlow, input-otp, Liveline, Leva, cmdk, Virtuoso, dnd kit, and Sonner.
