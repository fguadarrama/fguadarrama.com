# Minimal Landing

A static compact landing page built from the `compact-landing` skill style system.

## Files

- `index.html` — page content and structure
- `styles.css` — compact tactile UI system
- `script.js` — copy buttons, tabs, and status label behavior

## Edit

Replace these values first:

- `Minimal Landing`
- the tagline in `.tagline`
- `mailto:hello@example.com`
- footer email
- command text inside the `.command` button

## Deploy on GitHub Pages

Upload the three files at the root of your Pages repository, or keep them in a folder if your Pages path points there. No build step is required.
