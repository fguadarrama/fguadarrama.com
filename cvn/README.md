# Compact Francisco Landing

Static compact landing page for `fguadarrama.com`, designed as a minimal, compact-landing-style version of the existing public site.

## Files

- `index.html`
- `styles.css`
- `script.js`

## Deployment

Upload all files to the root of a GitHub Pages repository or any static host.

## Notes

- No build step and no dependencies.
- Uses Google Fonts: Geist and Geist Mono.
- The contact form validates input locally but does not send mail. Connect it to the same backend/form endpoint used by the current site before publishing it as a live contact form.
- Public content was transcribed from the live website on July 7, 2026.
